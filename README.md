# Learn AI 📚🧠

This repository documents my structured, personal journey to learn and master Artificial Intelligence — starting from Python programming and progressing through Machine Learning, Deep Learning, Generative AI, and MLOps.

My goal is to build a solid foundation by following a linear, book-based learning approach, applying knowledge through hands-on coding, notebooks, and mini-projects at every stage. This journey spans theory, implementation, and production-level AI systems.

---

## 🧭 Learning Roadmap

The journey is divided into six major stages:

### 1. Python Foundations
> **Focus:** Programming fundamentals, scripting, OOP  
> **Books:**  
> - *Learning Python* by Mark Lutz  
> - *Data Science from Scratch* by Joel Grus

### 2. Machine Learning
> **Focus:** Supervised & unsupervised learning, model evaluation, model tuning  
> **Books:**  
> - *An Introduction to Statistical Learning* (Python edition)  
> - *Hands-On Machine Learning with Scikit-Learn, Keras & TensorFlow*  
> - *Machine Learning with PyTorch and Scikit-Learn*  
> - *Regression and Other Stories* (optional, for statistical depth)

### 3. Time Series Forecasting
> **Focus:** Classical models (ARIMA, SARIMA), feature engineering, seasonality  
> **Books:**  
> - *Forecasting: Principles and Practice* by Hyndman and Athanasopoulos *(recommended)*  
> *(Alternate: Time Series Forecasting by Marco Peixeiro — more practical, less academic)*

### 4. Deep Learning
> **Focus:** ANN, CNN, RNN, model training and tuning, from-scratch understanding  
> **Books:**  
> - *Deep Learning with Python* by François Chollet  
> - *Deep Learning from Scratch* by Seth Weidman  
> - *Deep Learning* by Goodfellow, Bengio, Courville *(reference/theory-heavy)*

### 5. Generative AI & LLMs
> **Focus:** Transformers, LLMs, RAG pipelines, agents, prompt engineering  
> **Books:**  
> - *Natural Language Processing with Transformers*  
> - *Generative Deep Learning* by David Foster  
> - *Building LLMs for Production* by Peter Welinder, et al.

### 6. MLOps & System Design
> **Focus:** Scalable ML systems, deployment, monitoring, CI/CD, production-ready pipelines  
> **Books:**  
> - *Designing Machine Learning Systems* by Chip Huyen  
> - *Practical MLOps* by Noah Gift
