# Phase 2: Mathematics & Statistics

**Goal:** Strengthen mathematical and statistical foundations required for Machine Learning and Data Science.

---

## Resources

- **[Mathematics for Machine Learning and Data Science – DeepLearning.AI](https://www.coursera.org)** (Coursera Specialization)  
- **[Mathematics for Machine Learning – Marc Peter Deisenroth](https://drive.google.com/file/d/1SuUx37vogBZOQCEjvrDAp7SNUjI-YKhw/view?usp=drive_link)**  
- **[Practical Statistics for Data Scientists – Peter Bruce](https://drive.google.com/file/d/1Aj6lS7aatM9XZJDwY0KF441cBXw0ZY2k/view?usp=drive_link)**

---

## Learning Focus

- Linear algebra and matrix operations  
- Calculus for optimization and backpropagation  
- Probability, statistics, and data distributions  
- Hypothesis testing, correlation, and regression basics  

---

## Completion Goals

- Build strong mathematical intuition for ML models  
- Implement math concepts in Python notebooks  
- Apply statistical reasoning to real-world datasets
