# Generative AI Experiments

This folder contains hands-on experiments with **transformers, LLMs, and generative AI models**. Each project demonstrates building, fine-tuning, or deploying generative AI models.

## 📂 Project List

### Sample Project
- **Phase:** Deep Learning & LLMs  
- **Goal:** Experiment with text generation, embeddings, or question-answering systems  
- **Tech Stack:** Python, PyTorch, Hugging Face Transformers, LangChain  
- **Contents:**  
  - `notebooks/` – Notebooks for experimentation, training, and evaluation  
  - `models/` – Saved model weights and checkpoints  
  - `README.md` – Instructions for running and reproducing the project

---

## 📖 Guidelines
- Maintain clean and reproducible notebooks  
- Store model checkpoints in the `models/` folder  
- Add future generative AI mini-projects in the same folder structure
