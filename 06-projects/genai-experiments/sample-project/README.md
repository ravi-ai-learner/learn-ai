# Sample Project – Generative AI

This is a **sample Generative AI project** demonstrating experiments with transformers, embeddings, or LLMs.

## 📂 Structure

- `notebooks/` – Notebooks for training, inference, and evaluation  
- `models/` – Saved model weights and checkpoints  
- `README.md` – Instructions and overview of the project

## 🎯 Objectives

- Fine-tune or experiment with a transformer/LLM model  
- Implement text generation, summarization, or question-answering tasks  
- Store and version model checkpoints for reproducibility

## 🛠️ Tech Stack

- Python, PyTorch, Hugging Face Transformers, LangChain

## ✅ Completion Criteria

- Notebook reproduces all experiments end-to-end  
- Models are stored with versioning  
- Code is modular and ready for extension to new experiments
