# MLOps Deployments

This folder contains **production-ready ML and Generative AI deployment projects**. Projects demonstrate deploying, monitoring, and scaling models using modern MLOps tools.

## 📂 Project List

### Sample Project
- **Phase:** AI Engineering & MLOps  
- **Goal:** Deploy a trained ML or LLM model as an API or production pipeline  
- **Tech Stack:** Python, FastAPI, Docker, MLflow  
- **Contents:**  
  - `app/` – FastAPI application code  
  - `models/` – Serialized model files  
  - `README.md` – Instructions for running the deployment

---

## 📖 Guidelines
- Each project should demonstrate a **complete pipeline**: model → API → deployment  
- Maintain versioned models and reproducible deployment scripts  
- Add future MLOps projects using the same folder layout
